Open Connectik
📱 Install on Android
  1. 1Tap the menu ⋮ in the top-right of Chrome
  2. 2Choose Install app (or Add to Home screen)
  3. 3Tap Install — Connectik appears on your home screen
🍎 Install on iPhone / iPad
  1. 1Open this page in Safari (Chrome works on iOS 16.4+)
  2. 2Tap the Share icon (bottom of Safari)
  3. 3Scroll down → tap Add to Home Screen
  4. 4Tap Add — done!
💻 Install on desktop
  1. 1In Chrome or Edge, click the install icon in the address bar
  2. 2Click Install
Just open it in the browser →

Connectik is a web app (PWA) — no app store needed. Works on Android, iPhone, and desktop. On iOS 16.4+, install to your home screen to receive notifications.
